**This Repo Contains the Advanced Backend Infrastructure of a Large Scalable Social Media Web-Application**

***A User -> Login, Logout, Register***

***A User -> create a Post, Edit a Post, Like a post, Unlike a post , Comment on the post, Edit the Comment, Delete the Comment***

***Security Concerns are taken in consideration thus there is User-based-Authentication implementation using the middleware and only an Authenticated user can make a post, edit his/her post, delete his/her post***

***Authenticated user can make a commnet on a post, edit his/her post, delete his/her post***

**Everything is taken in consideration even the cascading deletes when a post is deleted then all the related comments and likes of this post will authomaticatlly be deleted**
